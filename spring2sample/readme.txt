Spring 2 Sample Application

author: Erwin Vervaet

Illustrates Spring MVC & Spring Web Flow. The same use-cases are implemented in 3
different ways so it's easy to compare:
* 'classic': Classic Spring 1.x based Spring MVC
* 'nextgen': New Spring 2.x based Spring MVC
* 'flow': New Spring 2.x based Spring MVC in combination with SWF

Compile with Java 5.
Required jars in WEB-INF/lib:
* commons-logging
* spring
* jstl
* standard (taglib)
* spring-binding
* spring-webflow
* ognl
